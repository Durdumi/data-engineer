# Stage 1 Fundamentals – ETL Pipeline

Учебный проект, демонстрирующий базовую архитектуру ETL-пайплайна
(Extract → Transform → Load) с продакшн-подходом.

Проект ориентирован на развитие навыков Data Engineer.

---

## Описание

Пайплайн:
- извлекает данные (mock-источник)
- применяет бизнес-логику (фильтрация по возрасту)
- загружает результат
- логирует каждый этап
- покрыт unit-тестами

---

## Архитектура

ETL разделён на независимые этапы:

- **Extract** — получение данных  
- **Transform** — бизнес-логика  
- **Load** — сохранение / вывод результата  
- **Orchestration** — управление пайплайном  

Точка входа в приложение — `main.py`.
